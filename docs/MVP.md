# StandUpReminder

StandUpReminder is a lightweight Windows 10 desktop application that remind user to take regular breaks while working at his PC. 
The application runs in tray and tracks user activity - pressing keys on keyboard, moving computer mouse or running video player. 

## Project Goals

* Provide simple, distraction-free break reminders.
* Minimize system resource usage.
* Integrate with Windows 10.
* Serve as a personal portfolio project.

## Planned Features

* System tray integration
* Native Windows notifications
* Configurable reminder intervals
* Automatic pause during user inactivity
* Persistent user settings
* Optional application auto-start

## Technology Stack

* C#
* .NET 8 (LTS)
* WPF
* JSON for configuration storage
* Git & GitHub
