# MVP 1.0

## Objective

Develop a lightweight Windows application that reminds users to take regular breaks while working at their computer.

## Functional Requirements

* Run as a Windows 10 desktop application.
* Minimize to the system tray.
* Display a native Windows notification every 30 minutes of active computer usage.
* Allow the reminder to be postponed for 5 minutes.
* Restart the timer after the user confirms the break.

## Configuration

* Reminder interval
* Enable or disable application auto-start
* Enable or disable notification sounds
* Active working hours

## Non-Functional Requirements

* Low memory usage
* Fast startup
* No administrator privileges required
